# Python Todo List (CLI)

A command-line Todo List application built in Python using file handling
and error handling concepts.

## Features
- Add tasks dynamically (no hard-coded numbers)
- View all tasks
- Mark tasks as completed
- Delete tasks using task number or task name
- Persistent storage using file handling
- Proper error handling to avoid crashes

## Requirements
- Python 3.x

## How to Run
```bash
python todoList.py
